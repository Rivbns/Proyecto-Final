El Profe Kinki

Si aprender quieres; errar debes, sin dejar de avanzar

Descripción

El Profe Kinki es un bot de discord que tiene como objetivo ayudar a los estudiantes de matemáticas. Este puede simplificar/resolver ecuaciones, funciones, etc gracias al uso de sympy. Mientras que su función de graficar, de numpy y PLT. Finalmente, su ultima función es que, atravez de una IA, el bot revise las imágenes que le mandes y te diga que tipo de función es. Esto gracias al modelo que que se puede ver en el repositorio.

  • !resolver: Este comando le permite al bot de Discord resolver ecuaciones matemáticas o simplificar expresiones que le den los usuarios de manera automática. Dependiendo del problema ingresado, el bot determina si es una ecuación (si contiene =) o una expresión algebraica. Es útil para estudiantes o cualquier persona que necesite resolver problemas matemáticos rápidamente dentro de un servidor de Discord.
  • !graficar: Este comando le permite al bot de Discord graficar funciones matemáticas y obtener un análisis detallado sobre la función ingresada por los usuarios. Este comando es útil para visualizar funciones y entender sus características matemáticas sin necesidad de herramientas externas, todo dentro de un servidor de Discord.
  • !check: Este comando le permite al bot de Discord dar una prediccion a los usuarios del tipo de funcion que contiene la imagen que ellos enviaron, utilizando un modelo de clasificación entrenado en Keras. Este sistema es útil para las personas que no esten seguros del tipo de funcion que estan viendo en un grafico cualquiera, se puede usar con la imagen obtenida del !graficar o de la web. Es importante resaltar que al dar el comando, el bot le pedira que adjunte una imagen y un nombre para la misma, para poder hacer la prediccion. Es preferible que la imagen sea de un grafico de una funcion matematica, por que no tendria sentido cualquier otra imagen.
